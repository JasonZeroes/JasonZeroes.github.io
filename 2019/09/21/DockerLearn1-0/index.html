MAC Docker安装与简单使用

发表于 2019-09-21 | 次阅读

一.Docker简介

  Docker是一个开源的容器引擎,是使用Golang语言来进行开发的,基于Linux内核的CGroup,Namespace,Union FS等技术实现,对应用程序进行隔离,并且独立于宿主机和其他进程,这种运行时封装的状态称为容器.

  Docker并非是一个完整的操作系统,所有的容器共享操作系统,是在进程间进行隔离.相比于虚拟机而言,它不需要像虚拟机那样必须要建立在硬件支持的基础之上.因为Docker容器出色的性能和良好的可移植性,现在在企业中越来越多的被用到.

二.Docker下载与安装

①下载安装包

  到Docker的官网下载对应平台的软件,然后进行安装.(在这里就不赘述具体怎么操作了,网上的教程有很多)

②修改镜像源

  由于默认的镜像下载源是国外的,十分的缓慢,可以到阿里云上获取加速器,然后修改对应的配置即可.

三.测试

  在终端输入以下命令,查看自己Docker是否已经安装正确;

①docker -v or dcoker –version

②docker run hello-world
